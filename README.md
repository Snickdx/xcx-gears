# XCX Gear DB

Xenoblade Chronicles X high-level weapons, armor, schematics & farming guide — searchable database with PWA support.

## Local

```sh
python -m http.server 8080
# → http://127.0.0.1:8080
```

## Deploy

Push to `master` — GitHub Pages auto-deploys from the root:

```sh
git push origin master
```

Site: https://snickdx.github.io/xcx-gears/
